## 脆弱性の報告

セキュリティに関する脆弱性を発見された場合は、公開のIssueトラッカーなど公の場では報告しないでください。
以下のいずれかの方法でご報告をお願いいたします。

### GitHub Private Vulnerability Reporting（推奨）

GitHubリポジトリの "Security" タブにある "Private vulnerability reporting" 機能が有効になっている場合は、そちらからご報告ください。これにより、開発者と安全にやり取りを行うことができます。

### メールでの報告

上記の方法が利用できない場合は、以下のメールアドレスまで詳細をお送りください。

**Email:** sazanami@kaedesato.work

報告には以下の情報を含めていただけますと幸いです。

*   脆弱性の種類と潜在的な影響
*   再現手順（可能であればPoCコードやスクリーンショット）
*   影響を受けるバージョン

報告を受け取り次第、速やかに内容を確認し、対応方針をご連絡いたします。
報告いただいた脆弱性が解決されるまで、情報の公開はお控えください。

ご協力に感謝いたします。
